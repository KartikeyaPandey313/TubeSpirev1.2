## üìñ What is TubeSpire?

**TubeSpire** is a web app that lets you download YouTube videos and audio in the highest available quality ‚Äî fast, free, and with zero ads or tracking. It is not a proxy, archive, or streaming service, and does not support any sites other than YouTube.

**You can use TubeSpire directly in your browser ‚Äî no installation, no accounts, no YouTube API keys required.**

## üñ•Ô∏è Local Installation (For Developers & Learners)

> You do **not** need to install anything to use TubeSpire online. This is for those who want to run it locally for learning or development.

**Requirements:**

- Python 3.8+
- [FFmpeg](https://ffmpeg.org/)
- [yt-dlp](https://github.com/yt-dlp/yt-dlp)

**Setup:**

```bash
git clone https://github.com/KartikeyaPandey313/TubeSpirev1.2/.git
cd TubeSpire
pip install -r requirements.txt
# Make sure ffmpeg and yt-dlp are installed and in your PATH
python app.py
```

Then open [http://localhost:5000](http://localhost:5000) in your browser.
